Approfondimento: Single Page Application

Una SPA è un'architettura web dove il server invia un solo file HTML. Vue.js intercetta le azioni dell'utente e aggiorna il contenuto del DOM senza ricaricare la pagina.

  • Performance: Riduce il traffico dati tra client e server.
  • Fluidità: Navigazione istantanea (come un'app mobile).
