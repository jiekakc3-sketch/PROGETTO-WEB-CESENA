# Progetto Sistemi Web - Unibo
**Studente:** Stéphane

### Descrizione
Applicazione SPA (Single Page Application) realizzata con **Vue.js** e **Tailwind CSS**.
L'app permette la gestione CRUD di strumenti informatici con salvataggio in LocalStorage.

### Funzionalità
- Visualizzazione dinamica delle sezioni.
- Creazione, Modifica ed Eliminazione di dati.
- Persistenza dei dati nel browser.
- Animazioni CSS fluide.
